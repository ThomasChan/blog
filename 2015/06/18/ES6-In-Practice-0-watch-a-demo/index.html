live-server

live-server这个package，可以帮助我们快速建立一个可以live reload的静态服务器。

jspm

jspm是一个package manager，可以帮助我们快速建立ES6的使用环境。让我们来看一下官方文档的getting start

reddit api

这里用了一个jsonp package，来发起jsonp请求。我们使用jspm install npm:jsonp来安装。

使用reddit api: 只要在每个r后面加.json。比如: https://www.reddit.com/r/pics.json。然后还要传param: 'jsonp'的参数。

说到API，突然想起codeacademy上面好像又好多教学，赶紧去一下先。
